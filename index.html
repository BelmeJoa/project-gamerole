THE GAME OF THE MAFIA

Este juego es un juego casero que se juega con minimo 6 personas de manera presencial, esta pagina es para facilitar el uso de una parsona ajena para el control del juego.

VALORES DE LAS CARTAS

  1. 12 = MAFIA
  2. 1 de espadas = POLICIA
  3. 7 de espadas = DOCTOR
  4. 7 de basto = LA PUTA
  5. 3 = CARNICERO
  6. 4 = PUEBLO

PARTICIPANTES

Se reparten las cartas y se coloca (EN EL MISMO ORDEN QUE ESTAN EN PANTALLA) el numero de carta que le corresponde a cada uno sin que nadie vea.
